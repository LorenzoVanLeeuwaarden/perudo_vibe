# Requirements: Perudo Vibe v3.0 The Gauntlet

**Defined:** 2026-01-21
**Core Value:** Fast-paced endurance mode where players face sequential 1v1 duels against AI, carrying dice between matches until eliminated.

## v3.0 Requirements

Requirements for The Gauntlet milestone. Each maps to roadmap phases.

### Core Gauntlet Loop

- [x] **GAUN-01**: Player can start Gauntlet mode from main menu
- [x] **GAUN-02**: Player plays 1v1 duels against AI opponents
- [x] **GAUN-03**: Player's dice count carries over between duels (no healing)
- [x] **GAUN-04**: AI opponent always starts with 5 dice each duel
- [x] **GAUN-05**: AI difficulty escalates as streak grows (Turtle → Calculator → Shark)
- [x] **GAUN-06**: Streak counter displays during gameplay
- [x] **GAUN-07**: Game ends when player loses all dice
- [x] **GAUN-08**: Player can immediately restart after game over

### Duel Transition Screen

- [x] **TRAN-01**: Victory splash shows defeated opponent after winning a duel
- [x] **TRAN-02**: Round number announcement displays ("ROUND 3", "ROUND 4", etc.)
- [x] **TRAN-03**: Next opponent introduction with name and personality flair
- [x] **TRAN-04**: Animated transition sequence (Tekken/boxing style fight card)

### Leaderboard

- [x] **LEAD-01**: Global leaderboard stored in Cloudflare D1
- [x] **LEAD-02**: Player submits score with nickname (30 chars, alphanumeric)
- [x] **LEAD-03**: Leaderboard displays top 100 scores with nicknames
- [x] **LEAD-04**: Player sees personal best score tracking
- [x] **LEAD-05**: Score fraud limits reject impossible scores
- [x] **LEAD-06**: "Near you" section shows scores immediately above/below player's rank
- [x] **LEAD-07**: Daily leaderboard resets at midnight UTC with countdown timer

### Achievements

- [x] **ACHI-01**: Streak milestone achievements at 5, 10, 25, 50, 100 opponents defeated
- [x] **ACHI-02**: Achievement toast notification on unlock
- [x] **ACHI-03**: Achievements persist in localStorage
- [~] **ACHI-04**: Progress indicators show "X/Y to next achievement" during run — *Descoped: User preferred toast-only approach*
- [x] **ACHI-05**: Hidden challenge achievements (e.g., "Win duel with only 1 die remaining")
